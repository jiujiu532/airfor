// turbo-all

1. Install Go dependencies
```
cd d:\Python\airforce\gpt-load && go mod tidy
```

2. Build the project
```
cd d:\Python\airforce\gpt-load && go build -o gpt-load.exe .
```

3. Run the project
```
cd d:\Python\airforce\gpt-load && make run
```
